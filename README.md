# BadApplePico8
Playing Toho Bad Apple on pico8 fantasy consol.

# Usage
Run bad_apple.p8 on pico8, drag and drop the frameData.bin onto the window.
# Build
frameData.bin is made using the python script styleToLua.py and converts the style.css sheet from this project [here](https://github.com/kevinjycui/css-video/tree/bb55bfc7f54495df760a0737c579e041763bba95)
